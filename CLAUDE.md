## Daft 0.7.2 API 备忘

- 无 `rename()` 方法，用 `select(col("old").alias("new"))` 替代
- `@daft.udf` 已废弃，使用 `@daft.func`（逐元素）或 `@daft.cls`（批处理）
- `explain()` 返回 `None`（直接打印到 stdout），需用 `io.StringIO` 捕获
- `fill_null()` 直接在 Expression 上调用
- `with_columns({})` 支持字典形式批量添加/修改列
